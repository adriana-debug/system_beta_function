Getting Started Guide

Welcome to the Enterprise Platform. This comprehensive guide will help you integrate with our APIs, configure your environment, and deploy your first application in minutes.

Prerequisites

Before you begin, ensure you have the following installed and configured:

  • Node.js 18.x or higher with npm/yarn package manager
  • Valid API credentials with appropriate access scopes
  • Command-line interface tools (CLI) version 2.4.0+
  • Administrative access to your organization's dashboard

Quick Start Installation

npm install @enterprise/platform-sdk
export PLATFORM_API_KEY="your_api_key_here"
npx platform init --template=starter

This command initializes a new project with our recommended starter template, including authentication, database connectivity, and basic API integration examples.

Authentication Setup

Our platform uses OAuth 2.0 with JWT tokens for secure authentication. Follow these steps to configure your application:

1

Generate API Credentials

Navigate to Settings → API Keys and create a new credential pair

2

Configure Environment Variables

Add your credentials to .env file or environment configuration

3

Initialize SDK Client

Import and configure the SDK with your credentials

First API Request

Here's a basic example of making your first authenticated request:

import { PlatformClient } from '@enterprise/platform-sdk';

const client = new PlatformClient({
  apiKey: process.env.PLATFORM_API_KEY,
  environment: 'production'
});

const response = await client.users.list({
  limit: 10,
  orderBy: 'created_at'
});

Rate Limiting

All API requests are subject to rate limiting. Standard tier allows 1,000 requests per hour. Enterprise plans have custom limits.
